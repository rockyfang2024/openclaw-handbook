# 1.4 连接服务器

**Mac / Linux 用户：用自带终端**
**Windows 用户：下载 [MobaXterm](https://mobaxterm.mobatek.net/) 或使用 PowerShell**

---

## 🔗 连接命令

```bash
ssh ubuntu@你的服务器IP
```

示例：
```bash
ssh ubuntu@123.45.67.890
```

第一次连接会提示：
```
Are you sure you want to continue connecting (yes/no)?
```

输入 `yes`，然后输入密码。

> 💡 输入密码时屏幕不会显示任何字符，这是正常的 Linux 安全机制，输入完直接回车即可。

---

## ✅ 连接成功后

你将看到类似界面：

```
Welcome to Ubuntu 22.04.3 LTS (GNU/Linux 5.15.0-91-generic x86_64)

Last login: Mon Apr  6 12:00:00 2026 from 123.45.67.890
ubuntu@VM-0-14-ubuntu:~$
```
