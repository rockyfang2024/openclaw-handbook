# 1.3 快速配置

**购买完成后，先做这些基础配置**

---

## 🔥 开放必要端口

在腾讯云控制台找到你的服务器 → 防火墙 → 添加规则：

| 类型 | 端口 | 用途 |
|------|------|------|
| SSH | 22 | 服务器登录 |
| HTTP | 80 | Web 界面访问 |
| HTTPS | 443 | 安全 Web 访问 |
| OpenClaw | 18789 | OpenClaw 默认端口 |

> ⚠️ **端口 22 是 SSH 登录端口，不要删除！**

---

## 🔄 更新系统软件

连上服务器后，第一件事更新软件：

```bash
sudo apt update && sudo apt upgrade -y
```

---

## 🛠️ 安装基础软件

```bash
# 安装 Node.js 18.x
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装 pnpm
npm install -g pnpm

# 安装 Git
sudo apt install git -y

# 安装 Docker（可选，后续部署用到）
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker ubuntu
```

---

## ✅ 验证安装

```bash
node --version    # 应显示 v18.x.x
npm --version    # 应显示 9.x.x
git --version    # 应显示 2.x.x
docker --version # 应显示 24.x.x
```
