# 第四章：创建定时任务

**目标：设置自动化提醒与 Cron 任务，让 Agent 主动推送**

---

## 1. 定时任务原理

| 概念 | 说明 |
|------|------|
| **Cron 表达式** | 定义执行时间 |
| **脚本文件** | 实际执行的任务 |
| **Channel** | 推送渠道（飞书/微信） |

文件位置：`~/.openclaw/cron-scripts/`

---

## 2. Cron 表达式入门

| 表达式 | 含义 |
|--------|------|
| `0 9 * * *` | 每天 9:00 |
| `0 9 * * 1-5` | 工作日 9:00 |
| `0 */2 * * *` | 每 2 小时 |
| `30 7 * * *` | 每天 7:30 |

格式：`分 时 日 月 周`

不会写？用 [crontab.guru](https://crontab.guru/) 在线验证。

---

## 3. 创建早安提醒

**第一步：创建脚本**
```bash
mkdir -p ~/.openclaw/cron-scripts
nano ~/.openclaw/cron-scripts/morning-quote.py
```

填入：
```python
#!/usr/bin/env python3
import random

quotes = [
    "早安！新的一天，继续加油 💪",
    "今日事，今日毕。开始行动吧！",
    "保持专注，持续迭代。",
]

print(random.choice(quotes))
```

**第二步：赋予权限**
```bash
chmod +x ~/.openclaw/cron-scripts/morning-quote.py
```

**第三步：配置定时执行**
```bash
nano ~/.openclaw/config.yaml
```

添加：
```yaml
cron:
  morning:
    script: "~/.openclaw/cron-scripts/morning-quote.py"
    schedule: "0 7 * * *"
    channel: feishu
    target: "你的飞书open_id"
```

**第四步：生效**
```bash
openclaw gateway restart
```

## 4. 常用提醒场景

```yaml
cron:
  morning:
    script: "脚本路径"
    schedule: "0 7 * * *"      # 每天 7:00
  
  lunch:
    script: "脚本路径"
    schedule: "0 12 * * *"     # 每天 12:00
  
  evening:
    script: "脚本路径"
    schedule: "0 22 * * *"     # 每天 22:00
  
  weekday:
    script: "脚本路径"
    schedule: "0 9 * * 1-5"    # 工作日 9:00
```

## 5. 查看和管理

```bash
openclaw cron list              # 查看定时任务
openclaw gateway logs | grep cron  # 查看执行日志
```
