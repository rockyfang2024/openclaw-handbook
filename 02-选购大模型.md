# 第二章：选购大模型

**目标：获取 AI 模型 API Key，让 Agent 具备智能**

---

## 1. 主流模型对比

| 模型 | 优势 | 劣势 | 推荐场景 |
|------|------|------|----------|
| **MiniMax** | 中文好、价格便宜、响应快 | 生态相对新 | ✅ 国内首选 |
| **DeepSeek** | 性价比高、开源 | 品牌认知度还在建立 | 追求性价比 |
| **Claude** | 英文能力强 | 国内访问稍慢 | 英文场景 |
| **GPT-4** | 通用能力强 | 成本高 | 预算充足 |

**推荐：MiniMax**（中文优化好 + 价格实惠 + OpenClaw 原生支持）

---

## 2. 注册 MiniMax

1. 打开 [MiniMax Platform](https://platform.minimaxi.com/)
2. 用手机号注册 → 完成实名认证
3. 进入控制台 → 「API Key」→ 创建
4. **立刻复制保存 Key**（关闭页面后无法再次查看）

建议充值 ¥10 测试（约 ¥1/百万 Token，便宜够用）。

---

## 3. 配置 OpenClaw 使用模型

编辑配置：
```bash
nano ~/.openclaw/config.yaml
```

填入：
```yaml
providers:
  minimax:
    api_key: "你的API Key"
    model: "MiniMax-M2"

default_provider: minimax
```

保存（Ctrl+X → Y → Enter）。

---

## 4. 验证配置

```bash
openclaw gateway restart
openclaw chat
```

能正常回复即配置成功。
