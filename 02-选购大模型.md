## 4. 配置 OpenClaw 使用模型

进入腾讯云服务器 OpenClaw 页面，点击左上角「配置」：

![点击配置按钮](.gitbook/assets/02-openclaw-config-btn.jpg)

点击配置后，选择模型（Model） → 选择 **Coding Plan** → 选择 **MiniMax Token Plan**，然后粘贴你的 API Key：

![模型配置页面](.gitbook/assets/02-model-config.jpg)
