## 4. 配置 OpenClaw 使用模型

编辑配置：
```bash
nano ~/.openclaw/config.yaml
```

填入：
```yaml
providers:
  minimax:
    api_key: "你的API Key"
    model: "MiniMax-M2"

default_provider: minimax
```

保存（Ctrl+X → Y → Enter）。

---

## 5. 验证配置

```bash
openclaw gateway restart
openclaw chat
```

能正常回复即配置成功。
