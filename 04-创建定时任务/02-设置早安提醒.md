# 4.2 设置早安提醒

**让 Agent 每天早上自动给你发消息**

---

## 📝 创建脚本

在服务器上创建脚本文件：

```bash
mkdir -p ~/.openclaw/cron-scripts
nano ~/.openclaw/cron-scripts/morning-quote.py
```

填入以下内容：

```python
#!/usr/bin/env python3
import random

quotes = [
    "早安！新的一天，继续加油 💪",
    "今日事，今日毕。开始行动吧！",
    "保持专注，持续迭代。",
]

message = random.choice(quotes)
print(message)
```

保存退出。

---

## 🔧 赋予执行权限

```bash
chmod +x ~/.openclaw/cron-scripts/morning-quote.py
```

---

## 🧪 测试脚本

```bash
python3 ~/.openclaw/cron-scripts/morning-quote.py
```

能正常输出名言即成功。

---

## ⏰ 配置定时执行

编辑 OpenClaw 配置：

```bash
nano ~/.openclaw/config.yaml
```

添加：

```yaml
cron:
  morning:
    script: "~/.openclaw/cron-scripts/morning-quote.py"
    schedule: "0 7 * * *"    # 每天 7:00
    channel: feishu          # 推送到飞书
    target: "你的飞书open_id"
```
