## 📖 七章学习路径

### [第一章：选购腾讯云服务器](./01-选购腾讯云服务器.md)

**目标：拥有自己的公网服务器，24小时运行 OpenClaw**

- 为什么需要云服务器
- 购买流程（腾讯云轻量应用服务器）
- 地域推荐新加坡，系统 Ubuntu 22.04 LTS

---

### [第二章：选购大模型](./02-选购大模型.md)

**目标：获取 AI 模型 API Key，让 Agent 具备智能**

- 主流模型对比（MiniMax / DeepSeek / Claude / GPT-4）
- 注册 MiniMax（邀请码享 9 折优惠）
- 购买 Plus-极速版 Token Plan（量大管饱，无 token 焦虑）
- 在 OpenClaw 页面配置模型
- 验证配置：问「你现在用的是什么大模型」→ 回复 MiniMax M2.7 即成功

---

### [第三章：接入飞书](./03-接入飞书.md)

**目标：在飞书中与龙虾机器人对话，随时随地唤醒 AI**

- 在 OpenClaw 页面授权接入飞书
- 创建飞书账号（如无）
- 开始在飞书中与龙虾机器人对话

---

### [第四章：创建定时任务](./04-创建定时任务.md)

**目标：让龙虾机器人按时提醒你做事**

- 最简方式：跟龙虾机器人说「几点提醒我做什么事」
- 推荐方案：本地 Cron + Python（稳定不失败）

---

### [第五章：创建 Skill](./05-创建Skill.md)

**目标：打造属于你自己的 AI 能力扩展**

- Skill 是什么（Agent 的能力扩展包）
- 创建「每日开心事」Skill：触发词 `每日开心事 xxxx`
- 高级进阶：配置 SSH，自动同步记录到 GitHub

---

### [第六章：配置多 Agent](./06-配置多Agent.md)

**目标：搭建多角色协作的 Agent 矩阵**

- 为什么需要多 Agent（一个团队，各司其职）
- 在飞书开放平台创建新应用（机器人 + 权限配置 + 事件回调）
- 让 OpenClaw 创建新 Agent（发送提示词 → 拿配对码 → 配对完成）
- 开始与新 Agent 对话

---

### [第七章：应用场景 & 案例](./07-应用场景.md)

**目标：感受 OpenClaw 在真实场景中的强大能力**

- 案例1：总结微信文章（自动归档 GitHub）
- 案例2：下载 GitHub 开源项目并改造优化
- 案例3：管理 GitHub 仓库，进行打卡记录
- 案例4：创建机器人群，增加聊天氛围
- 案例5：记录日常开销
- 案例6：输出技术文档（AI 协助编写）
