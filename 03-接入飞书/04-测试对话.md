# 3.4 测试对话

**验证飞书和 OpenClaw 是否成功对接**

---

## 🔄 重启 OpenClaw

```bash
openclaw gateway restart
```

查看日志确认飞书插件加载成功：

```bash
openclaw gateway logs
```

看到类似输出表示成功：
```
✅ Feishu plugin loaded
✅ Bot info: 我的AI助手
```

---

## 💬 测试对话

1. 打开飞书 App
2. 找到你的机器人（在「应用」里搜索应用名称）
3. 点击「发消息」
4. 输入：「你好」
5. 如果机器人回复了，配置成功！🎉

---

## ❓ 常见问题排查

**Q：发消息没反应？**
→ 检查 OpenClaw 是否启动：`openclaw gateway status`
→ 检查日志：`openclaw gateway logs`
→ 确认飞书回调 URL 能通：在本地浏览器访问 `http://你的IP:18789/health`

**Q：提示「应用不具备权限」？**
→ 飞书应用权限未审批完成，需要管理员审批

**Q：消息发送失败？**
→ 检查 App ID 和 App Secret 是否正确
→ 确认机器人能力已添加
