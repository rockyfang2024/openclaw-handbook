# 1.1 环境准备

**预计耗时：2 分钟**

---

## 🧰 需要安装的东西

| 软件 | 版本要求 | 用途 |
|------|----------|------|
| Node.js | ≥ 18.x | 运行 OpenClaw |
| npm / pnpm | 最新版 | 包管理器 |
| Git | 任意版本 | 代码管理 |

> 💡 **推荐使用 pnpm**，安装更快、占用更少。

---

## 📥 安装步骤

### 第一步：安装 Node.js

**macOS / Linux：**
```bash
# 使用 nvm 安装（推荐）
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
nvm install 18
nvm use 18
```

**Windows：**
直接去 [Node.js 官网](https://nodejs.org/) 下载 18.x LTS 版本，安装包点点点就完成了。

### 第二步：安装 pnpm（推荐）

```bash
npm install -g pnpm
```

### 第三步：安装 Git

**macOS：**
```bash
# 如果没有 git，执行：
xcode-select --install
```

**Ubuntu / Debian：**
```bash
sudo apt update && sudo apt install git
```

**Windows：**
去 [Git 官网](https://git-scm.com/) 下载安装。

---

## ✅ 验证安装

打开终端，执行以下命令：

```bash
node --version   # 应显示 v18.x.x
npm --version    # 应显示 9.x.x 或 10.x.x
git --version    # 应显示 git version 2.x.x
```

全部显示版本号即表示安装成功。

---

## ❓ 常见问题

**Q：提示 `command not found`？**
→ 重启终端，或重新打开一个新的终端窗口。

**Q：npm install 特别慢？**
→ 使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
```
