# 2.1 主流模型对比

**OpenClaw 支持多种大模型，选择适合你的**

---

## 📊 主流模型对比

| 模型 | 优势 | 劣势 | 推荐场景 |
|------|------|------|----------|
| **MiniMax** | 中文优化好，价格便宜，响应快 | 生态相对新 | ✅ 国内首选 |
| **DeepSeek** | 性价比高，开源能力强 | 品牌认知度还在建立 | 追求性价比 |
| **Claude** | 英文能力强，长文本理解好 | 国内访问稍慢 | 英文场景 |
| **GPT-4** | 通用能力强，生态成熟 | 成本高 | 预算充足 |

---

## 💡 我的推荐

**国内用户 → MiniMax**
- 中文理解好
- 价格实惠
- OpenClaw 原生支持

**进阶用户 → DeepSeek + MiniMax 组合**
- 主力用 MiniMax
- 特殊任务切换 DeepSeek

---

## 📝 如何获取 API Key

以 MiniMax 为例：

1. 访问 [MiniMax Platform](https://platform.minimaxi.com/)
2. 注册账号（手机号）
3. 进入控制台 → 创建 API Key
4. 复制 Key（格式：`eyJhbGc...`）

> ⚠️ **重要：API Key 只有第一次显示完整，之后看不到！先保存好。**
