# 2.2 MiniMax 注册与配置

---

## 📝 注册账号

1. 打开 [MiniMax Platform](https://platform.minimaxi.com/)
2. 点击「注册」，用手机号注册
3. 完成实名认证（国内合规要求）

---

## 🔑 创建 API Key

1. 登录后进入控制台
2. 左侧菜单 → 「API Key」
3. 点击「创建 API Key」
4. 输入 Key 名称（如 `openclaw`）
5. 点击确认，**立刻复制保存**

> ⚠️ 关闭页面后无法再次查看，只能新建。

---

## 💰 充值

MiniMax 有免费额度，但建议先充少量（如 ¥10）测试：

1. 控制台 → 充值
2. 选择支付方式
3. 确认金额

> 💡 MiniMax 价格参考：MiniMax-M2 模型约 ¥1/百万Token，便宜够用。
